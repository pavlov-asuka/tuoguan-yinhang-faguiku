# 商业银行托管业务法规库使用说明

## 1. 法规库简介

本法规库面向商业银行托管业务条线，覆盖公募基金、私募基金、证券期货资管、银行理财、信托、保险资金、养老金社保年金、跨境产品、REITs、衍生品、客户资金等托管相关法规和市场规则。

它的用途不是替代法律意见，而是为本地 agent 提供一套可检索、可核验、可维护的法规依据链。回答问题时，AGENT会优先使用本地库，再按需联网核验最新性和官方来源。

## 2. 使用指南：让本地 AGENT指定本地法规库为项目文件夹回答问题

将整个法规库文件夹下载到本地后，用支持读取本地文件的 AGENT打开本项目目录，AGENT会默认遵循根目录中的AGENTS.md的要求回答问题

适合提问的方式：

- `商业银行托管业务监督管理办法对托管职责怎么规定？`
- `基金托管人在估值复核中有哪些职责？`
- `银行理财产品托管涉及哪些核心规则？`
- `某条规则是否现行有效？本地库有没有官方来源？`

## 3. 目录说明

- `00_入口与索引/`：HTML 总目录、更新记录和待核验清单 `unresolved.md`。
- `01_法规原文库/`：官方网页、PDF、DOCX 等原始材料，是最终证据源。
- `02_文本抽取库/`：从原文抽取的可检索文本。
- `03_元数据台账/`：规则编号、名称、状态、来源、路径、优先级、产品标签、条线标签、市场标签和目录挂接。
- `tools/`：法规库构建、搜索索引构建和本地查询脚本。
