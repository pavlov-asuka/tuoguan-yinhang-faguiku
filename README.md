# 商业银行托管业务法规库使用说明

## 1. 法规库简介

本法规库面向商业银行托管业务条线，覆盖公募基金、私募基金、证券期货资管、银行理财、信托、保险资金、养老金社保年金、跨境产品、REITs、衍生品、客户资金等托管相关法规和市场规则。

本库以 `中国商业银行托管业务条线大法规库总目录.md` 为目录和标签来源。同一法规原文只保存一份，再通过产品标签、条线标签、市场标签和目录挂接复用。旧公募基金托管法规库已归档到 `98_历史归档/2026-06-12_公募基金托管库/`，新库复用了其中已下载的原文和文本。

本库不是法律意见替代品。回答问题时，应优先使用本地库，再按需联网核验现行性和官方来源，并区分“原文依据”和“分析判断”。

## 2. 使用指南

适合提问的方式：

- `商业银行托管业务监督管理办法对托管职责怎么规定？`
- `基金托管人在估值复核中有哪些职责？`
- `银行理财产品托管涉及哪些核心规则？`
- `某条规则是否现行有效？本地库有没有官方来源？`

推荐查询链路：

1. 先用本地索引召回候选条款：

   ```powershell
   python tools\search_kb.py "关键词" --status 现行有效 --limit 8
   ```

2. 用 `03_元数据台账/rules_index.csv` 或 `rules_index.json` 核对法规名称、状态、优先级、官方来源和本地路径。
3. 对业务场景型问题，补查 `04_托管业务专题地图/托管业务专题地图.md`，避免只按关键词遗漏规则组。
4. 回看 `02_文本抽取库/` 的文本上下文，并回到 `01_法规原文库/` 和官方 URL 核验。
5. 如结论依赖但本地库未收录的规则、附件或配套文件，应登记到 `99_unresolved/unresolved.md`，并标注“待月度巡检”。

`search_kb.py` 支持常用筛选：

```powershell
python tools\search_kb.py "估值 复核" --product 公募基金 --line 核算估值 --status 现行有效
python tools\search_kb.py "登记结算" --market 银行间 --limit 10
python tools\search_kb.py "托管业务" --category A-02 --include-aux
```

## 3. 维护指南

完整重建：

```powershell
python tools\build_knowledge_base.py
```

该脚本会：

1. 解析大法规库总目录，过滤内部落库位、内部模板、样表、流程图、知识运营等非外部法规项；
2. 去重生成 `03_元数据台账/master_rules.json` 和 `rules_index.*`；
3. 复用历史归档中的旧库原文和文本；
4. 对可识别的 P0 单部法律尝试从国家法律法规数据库补源；
5. 刷新 HTML 总目录、专题地图、unresolved、旧库复用清单和 SQLite 搜索索引。

搜索索引：

- `03_元数据台账/search.db` 是由台账和文本抽取库生成的 SQLite FTS5 检索加速层。
- 它不是法规依据，可删除后用 `python tools\build_search_index.py` 重建。
- 结论仍应回到 `rules_index.*`、文本、原文和官方 URL 核验。

每次新增、替换、修正、扩展法规时，至少检查并按需同步：

- `README.md`
- `00_入口与索引/总目录.html`
- `00_入口与索引/更新记录.md`
- `03_元数据台账/master_rules.json`
- `03_元数据台账/rules_index.md`
- `03_元数据台账/rules_index.csv`
- `03_元数据台账/rules_index.json`
- `03_元数据台账/search.db`
- `04_托管业务专题地图/托管业务专题地图.md`
- `99_unresolved/unresolved.md`

## 4. 目录说明

- `中国商业银行托管业务条线大法规库总目录.md`：新库目录、标签和优先级口径来源。
- `00_入口与索引/`：HTML 总目录和更新记录。
- `01_法规原文库/`：官方网页、PDF、DOCX 等原始材料，是最终证据源。
- `02_文本抽取库/`：从原文抽取的可检索文本。
- `03_元数据台账/`：规则编号、名称、状态、来源、路径、优先级、产品标签、条线标签、市场标签和目录挂接。
- `04_托管业务专题地图/`：按大目录、产品、条线和市场组织规则。
- `98_历史归档/`：重构前旧公募基金托管库归档和复用来源。
- `99_unresolved/`：待核验、待扩展、来源不完整或不能作为正式依据的事项。
- `tools/`：法规库构建、搜索索引构建和本地查询脚本。
